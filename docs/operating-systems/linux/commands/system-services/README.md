# System Services

Learn how to manage Linux services, inspect system logs, and control the `systemd` init system using essential system service commands.

---

## 📖 Overview

Modern Linux distributions use **systemd** to manage system services, background processes (daemons), startup targets, and logging. Understanding how to control services and analyze logs is a fundamental skill for system administrators, DevOps engineers, and developers.

This section covers the essential commands used to start, stop, monitor, troubleshoot, and inspect Linux services.

---

## 🎯 What You'll Learn

Topics covered in this section include:

- Managing system services
- Starting and stopping services
- Enabling and disabling services at boot
- Viewing service status
- Inspecting system logs
- Understanding the systemd journal
- Troubleshooting service failures
