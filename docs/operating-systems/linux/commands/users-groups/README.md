# Users & Groups

Learn how to manage Linux users, groups, permissions, and account-related information using essential user and group management commands.

---

## 📖 Overview

Linux is a multi-user operating system where every file, process, and resource is owned by a user and associated with one or more groups. Understanding how users and groups work is essential for managing permissions, securing systems, and administering servers.

This section covers the core commands used to create, modify, inspect, and manage user accounts and groups.

---

## 🎯 What You'll Learn

Topics covered in this section include:

- Viewing user information
- Managing user accounts
- Creating and deleting groups
- Changing passwords
- Switching users
- Managing group memberships
- Understanding user and group IDs

## 📌 Quick Summary

- Linux uses users and groups to control access to system resources.
- Commands like `useradd`, `usermod`, and `userdel` manage user accounts.
- Commands such as `groupadd` and `groups` manage group memberships.
- `id`, `who`, and `su` help inspect and switch user contexts.
- Understanding users and groups is fundamental for Linux security and system administration.
