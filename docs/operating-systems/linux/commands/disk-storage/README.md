# Disk Storage

Learn how to monitor, analyze, and manage disk space, filesystems, and storage devices using essential Linux disk storage commands.

---

## 📖 Overview

Efficient storage management is a fundamental Linux administration skill. Whether you're monitoring server capacity, troubleshooting storage issues, or managing disks, Linux provides powerful command-line tools to inspect and analyze storage usage.

This section covers the core commands used to view filesystem statistics, analyze directory sizes, inspect storage devices, and manage mounted filesystems.

---

## 🎯 What You'll Learn

Topics covered in this section include:

- Monitoring disk usage
- Analyzing directory sizes
- Viewing storage devices
- Understanding mounted filesystems
- Identifying storage bottlenecks
- Troubleshooting low disk space

---

## 📂 Contents

| Document | Description |
|----------|-------------|
| `df` | Display filesystem disk space usage |
| `du` | Estimate file and directory disk usage |
| `fdisk` | Create and manage disk partitions |
| `lsblk` | List block storage devices |
| `mount` | Mount filesystems |
| `umount` | Unmount filesystems |

## 📌 Quick Summary

- Linux provides powerful tools for monitoring and managing storage devices and filesystems.
- `df` displays filesystem usage, while `du` analyzes the size of individual files and directories.
- `lsblk` lists available storage devices and partitions.
- `mount` and `umount` manage filesystem mounting.
- `fdisk` is used to create and modify disk partitions.
- Mastering these commands is essential for system administration, troubleshooting, and capacity planning.
