# Compression

Learn how to compress, decompress, archive, and extract files using essential Linux compression and archiving commands.

---

## 📖 Overview

Compression is a fundamental part of Linux system administration and software distribution. It helps reduce file sizes, save disk space, speed up file transfers, and package multiple files into a single archive.

Linux provides several tools for creating archives, compressing files, and extracting compressed data. Knowing when to use each tool is an essential skill for developers, system administrators, and DevOps engineers.

This section covers the most commonly used compression and archiving commands.

---

## 🎯 What You'll Learn

Topics covered in this section include:

- Compressing files
- Extracting archives

- Choosing appropriate compression formats

## 📌 Quick Summary

- Compression reduces file sizes to save storage space and improve transfer speeds.
- Archiving combines multiple files into a single package for easier distribution and backup.
- `tar` is the standard Linux archiving utility.
- `gzip` and `gunzip` compress and decompress files using the Gzip format.
- `zip` and `unzip` provide cross-platform archive compatibility.
