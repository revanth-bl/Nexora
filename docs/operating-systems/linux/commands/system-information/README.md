# System Information

Learn how to inspect your Linux environment, user session, system identity, and shell configuration using essential system information commands.

---

## 📖 Overview

Linux provides several commands to display information about the current user, operating system, environment variables, command history, and shell configuration. These commands are invaluable for troubleshooting, scripting, system administration, and understanding your working environment.

Whether you're checking who is logged in, examining your `PATH`, or viewing environment variables, these tools are part of every Linux user's daily workflow.

---

## 🎯 What You'll Learn

Topics covered in this section include:

- Identifying the current user
- Viewing operating system information
- Working with environment variables
- Creating command aliases
- Managing command history
- Understanding shell configuration

---

## 📂 Contents

| Document | Description |
|----------|-------------|
| `alias` | Create and manage command shortcuts |
| `env` | Display or modify environment variables |
| `export` | Export variables to child processes |
| `history` | View and manage shell command history |
| `uname` | Display operating system and kernel information |
| `whoami` | Show the current logged-in user |
