# Package Management

Install, update, remove, and manage software packages on Linux systems using various package managers.

---

## 📖 Overview

The **Package Management** section covers the tools and concepts used to manage software on Linux distributions.

Linux distributions use package managers to install applications, resolve dependencies, update software, and maintain system packages. Understanding package management is an essential skill for Linux users, system administrators, and DevOps engineers.

Since different distributions use different package managers, Nexora documents the most widely used ones across the Linux ecosystem.

---

## 🎯 What You'll Learn

Topics in this section include:

- Installing software packages
- Updating system packages
- Removing applications
- Searching package repositories
- Managing dependencies
- Verifying installed packages
- Working with multiple package managers
