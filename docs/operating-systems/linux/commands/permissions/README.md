# Permissions

Learn how Linux controls access to files and directories using permissions, ownership, and security principles.

---

## 📖 Overview

The **Permissions** section covers one of the most fundamental concepts in Linux: controlling who can access, modify, and execute files and directories.

Every file and directory in Linux has:

- An **owner**
- A **group**
- A set of **permissions**

Understanding how these work is essential for system administration, server management, cybersecurity, and everyday Linux usage.

---

## 🎯 What You'll Learn

Topics in this section include:

- File and directory permissions
- Read, write, and execute permissions
- Numeric (octal) permission notation
- Symbolic permission notation
- File ownership
- Group ownership
- Security best practices
