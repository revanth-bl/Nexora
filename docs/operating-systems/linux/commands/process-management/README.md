# Process Management

Learn how Linux creates, monitors, controls, and terminates running processes using essential process management commands.

---

## 📖 Overview

Every program running on a Linux system is a **process**. Linux provides a rich set of tools to monitor process activity, inspect resource usage, send signals, and terminate applications when necessary.

Whether you're a beginner troubleshooting a frozen application or a system administrator managing production servers, understanding process management is an essential Linux skill.

---

## 🎯 What You'll Learn

Topics covered in this section include:

- Viewing running processes
- Monitoring CPU and memory usage
- Understanding Process IDs (PIDs)
- Sending signals to processes
- Gracefully and forcefully terminating processes
- Managing multiple processes

---

## 📂 Contents

| Document | Description |
|----------|-------------|
| `ps` | Display information about running processes |
| `top` | Monitor processes in real time |
| `kill` | Send signals to a process using its PID |
| `killall` | Send signals to processes by name |
