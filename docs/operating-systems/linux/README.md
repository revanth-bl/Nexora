# 📖 Overview

Linux is one of the world's most widely used operating systems. It powers servers, cloud infrastructure, supercomputers, embedded devices, Android smartphones, and much of the modern internet.

Whether you're a student, software developer, DevOps engineer, cloud architect, cybersecurity professional, or system administrator, Linux is an essential skill.

This section of **Nexora** is designed to help you understand not just **how** Linux works, but **why** it works the way it does.

# 🚀 Getting Started

If this is your first time learning Linux, begin here:

➡️ **Commands → File Management → `pwd`**

Understanding where you are in the filesystem is the foundation for everything else you'll learn.

After that, continue with:

1. `ls`
2. `cd`
3. `mkdir`
4. `touch`
5. `cp`
6. `mv`
7. `rm`
